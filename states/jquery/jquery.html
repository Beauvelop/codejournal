THIS in two different contexts:

'this' can refer--here--to the element that fired this
...or the object created by the triggering of that element.

HOWEVER: What if I want to call the 'doIt()' method on that object
& refer to that element as as 'this'??

Simple: We use javascript's .call(); method.


  btn = $('button');
  var obj = {
    doIt: function(e) {
      console.log(this);
    }
  }

btn.on('click', function(e) {
  obj.doIt().call(this);
  //we call this b/c within this click function
  //we know that 'this' refers to the btn that was clicked.
})






I can use the call method & send another arg to the doIt function;



  btn = $('button');
  var obj = {
    doIt: function(e) {
      e.hide();
    }
  }
  btn.on('click', function(e) {
    obj.doIt.call(this, btn.siblings());
  })

  
This will cause all things but the button to hide

  


  How to execute code/animation after completion of one animation?

  var obj = {
