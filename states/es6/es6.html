The "let" keyword. var has 2 kinds of scope. Global and Lexical (or functional) scope.

This code doesn't throw an error in es5. 
var gimmeSum = function(sum) {
  if(sum) {
    var total = sum + 3;
  }
  return total;
}
alert(gimmeSum(7))
// 10

However, let variables are not implicity hoisted

var gimmeSum = function(sum) {
  if(sum) {
    let total= sum + 3;
  }
  return total;
} // undefined
