Centering with Translate:

This is really the best example.



.ctr-container-ex3 {
  position: relative;
  height: 40vh;
  width: 100%;
}
.ctr-example3 {
  height: 200px;
  width: 200px;
  border: 1px solid fuchsia;
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
}

This is centered both horiz & vertically!
